Listas na HTML5

Listas Ordenadas

Para criar Listas Ordenadas é necesário usar a tag ol e, dentro dela, inserir tags li que significam os itens dessa lista. Exemplo:

Ovo Frito

  1. Adicione uma frigideira ao fogo
  2. Na frigideira já quente, adicionar uma colher de manteiga
  3. Dissolver a manteiga em toda a superfície da frigideira
  4. Salpicar sal a gosto
  5. Color o ovo para fritar
  6. Após a parte inferior do ovo ficar corada, vire
  7. Quando o ovo e sua clara estiverem sólidos, retire do fogo
  8. Sirva

Listas Não Ordenadas

Para criar listas não ordenadas é só trocar ol por ul.

Ingredientes

  • Manteiga
  • 1 ovo
  • Sal
  • Orégano
  • Pimenta Calabresa

Listas de Definição

A criação de listas de definição é diferente. É necessário utilizar a tag df, dentro dela utilizar a tag dt e dentro da tag dt usar dd.
