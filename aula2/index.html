Um pouco de semântica...

Posso usar a tag bold para deixar textos em negrito. Entretanto, essa tag não possui nenhum significado especial, não é semântica.

Em contrapartida, é possível usar a tag italic para se fazer entender que o texto a seguir deve ser lido de maneira diferente. São palavras como: textos literários, termos técnicos, pensamentos, palavras estrangeiras... Capisce? Essa sim é semântica.

Agora, para deixar de maneira mais polida fórmulas e conceitos da área de exatas, se pode usar as tags sup e sub.

Sup numa fórmula matemática: Como Gravidade na Terra, pode-se assumir o valor 9 x 109.

Sub num Elemento Químico: H2O.

Um pouco mais do mesmo...

Para realizar abreviações de palavras, se usa a tag abbr. Como exemplo, NASA. A NASA faz umas coisas incríveis!

Para se fazer referẽncia a algum texto ou trabalho acadêmico, se usa a tag cite, a tag de citação curta. Como exemplo: Uma Breve História do Tempo, de Stephen Hawking , vendeu mais de 10 milhões de cópias em todo o mundo!

Quando se apresentar uma nova terminologia durante algum texto, é interessante fazer uso da tag defition. Como exemplo: Um Buraco Negro é uma região do espaço onde nada, nem luz, consegue escapar.

Para dar ênfase e importância máxima a algum conteúdo na página, é bom usar as tags strong e em para isso. Como exemplo para strong: CUIDADO! Homens trabalhando.

Agora, para em, que é usada para mudar o significado de uma frase:

Eu acho que a Ivy foi a minha primeira...

Para criar citações que usam um parágrafo inteiro, se usa a tag blockquote. Para citações que permanecem dentro de um parágrafo, se usa a quotation.

Exemplo blockquote:
Como dito por Red Harlow em Red Dead Revolver:

It never was about the money.

Exemplo de citação curta: Como bem dizia George Orwell: Ignorância é força!

Para dar informações de endereço de alguém, se usa a tag address. Exemplo:

Meu email

742, Evergreen Terrace, Springfield

Para alterações de conteúdo como inserção de novo texto, texto que foi deletado e alterações no texto que ainda devem ser mantidas, se usa ins para inserção, del para deletar e strikethrough para manter.

Exemplo ins e del:

Essa foi a pior melhor ideia que já teve.

Exemplo strikethrough:

Notebook Dell Super Compacto!

Era: R$1000,00

Agora sai por: R$500,00!
