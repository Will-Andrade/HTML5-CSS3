Marcação HTML5 Extra

DOCTYPE
Doctypes são usados para que o navegador compreenda qual versão da HTML estou usando no arquivo.

Atributos ID e Class
Atributos ID servem para adicionar a um elemento um identificador único que pode ser usado tanto num arquivo CSS3 quanto num arquivo JavaScript para manipulação de estilos e funcionalidades do elemento. O atributo Class também. A diferença é que enquanto só se pode usar uma ID para cada elemento, é possível anexar várias Classes a um elemento e atribuir a ele diversas funcionalidades e estilos dessa forma.

Inline Elements e Block Elements
Elementos Inline são elementos que podem ser inseridos dentro de elementos de bloco, ocupam apenas uma linha sem criar alguma nova e também ocupam apenas a largura necessária na tela. Em contrapartida, Elementos de Bloco fazem o contrário. Eles iniciam em nova linha, ocupam 100% da largura e não podem ser inseridos dentro de elementos inline.

Div e Span
A tag div é usada como agrupador quando não se possui outra alternativa disponível, sendo um agrupador de elementos de elementos de bloco. A tag span é um elemento inline e é usado quando se deseja alterar o estilo ou funcionalidade de uma pequena parte de alguma coisa (texto, por exemplo).

iFrames e Caracteres de Escape
iFrames são usados para incluir conteúdo externo na página. Como outras páginas, mapas, textos, etc. Caracteres de Escape são usados em conjunto com o símbolo "&" e lhe permitem inserir caracteres reservados da HTML5 no texto sem problemas.

Meta
As tags de metadados que são usadas na Head do HTML. Elas são necessárias para inserir funcionalidades como descrição doque a página contém em seu corpo, atributo description, palavras chave que o usuário pode digitar na pesquisa e encontrar a página, atributo keywords, se os mecanismos de busca devem ou não deixar essa página nos resultados de pesquisa, atributo robots, autores da página, atributo author, evitar que o browser armazene cache, atributo pragma e dar uma data de vencimento do cache, atributo expires. Os 3 primeiros são usados com o atributo name e os 3 últimos com o atributo http-equiv.
