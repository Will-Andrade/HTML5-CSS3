Formulários

Para criar um formulário, se usa a tag form e como atributo se usa:

  • Action que tem como valor a página de processamento do que o form estiver passando para o servidor;
  • Method que recebe como valor GET ou POST sendo GET para pegar info de um banco, por exemplo, e post para enviar info para um banco.
